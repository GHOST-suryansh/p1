## Projects Included

### 1. QR Code Generator

QR Code Generator is the main and most complete web app in this project.  
It allows users to generate QR codes from text or URLs instantly.

Features:
- Generate QR codes in real time  
- Supports text and URLs  
- Simple and clean user interface  
- Runs completely on the client side  

Technologies Used:
- HTML  
- CSS  
- JavaScript
- python  

This app is useful for quickly creating QR codes without using external tools.
